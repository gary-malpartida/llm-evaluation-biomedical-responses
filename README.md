# LLM Evaluation for Biomedical Responses

## Overview
This repository is designed to evaluate AI-generated responses in biomedical, molecular biology, genetics, and public health contexts using structured scoring criteria and domain-specific review standards.

## Objectives
- Assess the factual accuracy and completeness of biomedical responses generated by large language models.
- Develop a practical evaluation rubric for domain-specific quality review.
- Organize prompts, model outputs, and reviewer scores in a reproducible format.
- Explore common error patterns in AI-generated scientific content.

## Planned Repository Structure
- `data/` → prompts, model responses, and evaluation tables
- `rubric/` → scoring criteria and review guidelines
- `notebooks/` → exploratory analysis and scoring summaries
- `results/` → summary tables and visual outputs
