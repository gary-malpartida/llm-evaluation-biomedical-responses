# Evaluation Analysis Notebook (Plan)

## Objective
Analyze LLM-generated biomedical responses using structured evaluation scores.

---

## Planned Analyses

### 1. Score Distribution
- Visualize distribution of overall scores
- Identify high vs low performing responses

### 2. Dimension Analysis
- Compare factual accuracy vs clarity vs completeness
- Identify weakest evaluation dimension

### 3. Domain Performance
- Compare performance across:
  - molecular_biology
  - genetics
  - epidemiology
  - biochemistry
  - public_health

### 4. Error Patterns
- Identify common weaknesses in responses
- Classify errors into:
  - factual errors
  - incomplete reasoning
  - terminology misuse

### 5. Risk Analysis
- Evaluate safety/risk scoring trends
- Detect potentially unsafe outputs
